## User Installation Flow

When users clone from GitHub, they will:

```bash
# 1. Clone repository
git clone <your-repo-url>
cd finops-agent-cdk

# 2. Run deployment script (handles everything)
./scripts/deploy.sh
```

The deployment script automatically:
1. Prompts for admin email
2. Runs `npm install` (installs dependencies)
3. Runs `npm run build` (compiles TypeScript)
4. Bootstraps CDK if needed
5. Deploys all 3 stacks
